# Cabinet Dentaire Dr. Benjebara Ghali 🦷✨

A modern, high-performance, and responsive website built for the **Dr. Benjebara Ghali Dental Clinic**. The application features a premium UI design incorporating glassmorphism, fluid interactive animations, and full bilingual French/Arabic support with RTL (Right-to-Left) layout transitions.

## 🚀 Key Features

- **🌐 Dual-Language Support (FR/AR):** Built-in context provider offering seamless toggle between French and Arabic, automatically configuring `dir="rtl"` layout for Arabic script and persisting user preference via `localStorage`.
- **✨ Fluid Animations & Transitions:** Powered by `framer-motion` for reveal transitions and micro-interactions.
- **📜 Ultra Smooth Scroll:** Integration of `lenis` smooth scroll behavior for an elevated user experience.
- **🎨 Premium Visual Identity:** Fully customized CSS glassmorphic components using a custom teal & blue brand palette.
- **📈 Interactive UI Components:**
  - **Results Slider:** Before/After slider to showcase treatment results.
  - **Interactive Booking Form:** Client-side appointment request form.
  - **Why Us & Stats Counter:** Clear visual highlights of clinic key metrics and values.
  - **WhatsApp Floating Button & Map Integration:** Streamlined customer outreach options.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router, React 19)
- **Styling:** CSS Modules, Modern Vanilla CSS variables, and Google Fonts (Poppins, Montserrat, JetBrains Mono)
- **Animations:** Framer Motion, Lenis Smooth Scroll

## 📦 Getting Started

### Prerequisites

- Node.js (v18.x or later recommended)
- npm or yarn

### Installation & Run

1. Navigate to the website subdirectory:
   ```bash
   cd website

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:3000`.
